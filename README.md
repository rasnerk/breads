# Breads

## API

| Method        | Path           | Purpose |
| ------------- | -------------  | ------------- |
| GET           | /              | Renders welcome message |
| GET           | /breads        | Gets all breads data |
| POST          | /breads        | Creates a new bread |
| GET           | /breads/:index | Gets bread in index :index |
| GET           | /breads/new    | Renders form to create new bread |
